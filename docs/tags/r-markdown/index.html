On blogging with R

As noted previously, I use R for school and work when doing statistical computing using biomedical data. I tend to prefer sharing nicely formatted HTMLs generated from analyses conducted in R using R Markdown. Therefore, I was excited to learn that it was now possible to integrate “knitted” HTML output from an R Markdown (.Rmd) file to my Hugo blog with the assistance of the new R package, blogdown.
